\section{Tasks} 

During the elaboration phase we will do the following tasks:

\begin{enumerate}
    \item [\textbf{Formalization}] which consists in the formalization of the \gls{SSE} protocol to implement;
    \item [\textbf{Implementation}] consists in the implementation of the formalized protocols. At the end of this task we will have produced software artifacts;
    \item [\textbf{Evaluation}] will be done at the end of each implementation consists in the evaluation of the prototypes in terms of performance and security, as stated in Section \ref{ch3:sec:evaluation-guidelines};
    \item [\textbf{Validation}] will be done at the end of each implementation task and consists in testing the correctness of the prototyped solutions, as stated in Section \ref{ch3:sec:evaluation-guidelines};
    \item [\textbf{Writing}] the rest of thesis, which will be done in parallel with the other tasks.
\end{enumerate}
