\chapter{Work Plan}
\label{cha:work-plan}

\prependtographicspath{{Figures/}}

On this chapter we present the work plan for the elaboration phase, depicted in the following Gantt chart \ref{fig:work-plan}. During the elaboration phase we will do the following tasks:

\begin{enumerate}
    \item [\textbf{Protocol Design}] which consists in the designing the \gls{SSE} protocol to implement;
    \item [\textbf{Implementation}] consists in the implementation of the protocols. At the end of this task we will have produced software artifacts;
    \item [\textbf{Evaluation}] will be done at the end of each implementation consists in the evaluation of the prototypes in terms of performance and security, as stated in Section \ref{ch3:sec:evaluation-guidelines}; At the end of this task we will have produced graphs which describe the result metrics;
    \item [\textbf{Validation}] will be done at the end of each implementation task and consists in testing the correctness of the prototyped solutions, as stated in Section \ref{ch3:sec:evaluation-guidelines}; At the end of this task we will have produced graphs which describe the result metrics;
    \item [\textbf{Writing}] the rest of thesis, which will be done in parallel with the other tasks.
\end{enumerate}
